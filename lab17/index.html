OOP Design: Hangman

1. Understand the problem

Set up the game and include gallows and players
Prompt Player 1 for word
Display length of word
Prompt Player 2 for letter guess
Validate Player 2 input
Check if guess matches any of the letters within the word

2. Pseudocode the High Level Approach

Set up the game initially
  Create hangman gallows
  Create a players
	Render gallows
	Establish 6 lives for Player 2
	Prompt Player one for word
	Validate input and display number of letters as underscores
Start the game loop
	Prompt Player 2 for letter guess
	Validate guess and check with word
	If the guess matches
		Replace corresponding underscore with letter
		Keep looping
	Else
		Remove one life
		Keep looping
	If all letters are guessed
		Display Victory Message
		Stop looping
	If lives equal 0
		Display "Game Over"
		Stop Looping
